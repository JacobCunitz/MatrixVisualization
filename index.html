To generate a matrix, put in the desired rows and columns, and check the box if you would like random colors, or not for the matrix to be blank. Then click submit to generate the matrix. If the matrix has been randomized and you would like to sort the rows, click the "Click to Sort Rows" button to sort them. Trying to generate a new matrix will overwrite the previous one.
